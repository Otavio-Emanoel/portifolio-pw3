# Imagens dos Projetos

Adicione nesta pasta as imagens de preview dos seus projetos com os seguintes nomes:

- `project-portfolio.png` — Preview do Portifólio
- `project-api.png` — Preview da API REST
- `project-ecommerce.png` — Preview do E-commerce
- `project-dashboard.png` — Preview do Dashboard
- `project-mobile.png` — Preview do App Mobile
- `project-microservices.png` — Preview dos Microserviços

Recomendação: imagens de **1200×630px** ou **800×450px** em formato PNG ou WebP.
